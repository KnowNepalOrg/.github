# Know Nepal 🇳🇵

Know Nepal is an open-source organization building software, APIs, and open datasets that make information about Nepal more accessible, reliable, and reusable.

---

## Our Mission

To build open digital resources that support developers, researchers, students, organizations, and the wider community through open-source collaboration.

---

## What We Build

- Geography & Administrative Data
- Public Services
- Tourism & Culture
- Public APIs
- Open Datasets
- Developer Tools

---

## Technology

**Frontend**
- React
- Next.js
- TypeScript
- Tailwind CSS

**Backend**
- Java
- Spring Boot
- PostgreSQL
- REST APIs

## Contributing

Know Nepal is community-driven. Whether you're writing code, improving documentation, fixing bugs, or contributing datasets, your contributions are welcome.

Explore our repositories, pick an issue, and submit a pull request.

## Vision

To build a trusted open-source ecosystem for Nepal's digital knowledge.
