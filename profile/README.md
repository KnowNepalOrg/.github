# Know Nepal 🇳🇵

Know Nepal is an open-source organization building software, APIs, and open datasets that make information about Nepal more accessible, reliable, and reusable.

## Focus Areas

- Geography & Maps
- Tourism & Culture
- Public Services & Education
- Open Data & APIs
- Developer Tools & Infrastructure

## Technology

**Frontend:** React, Next.js, TypeScript, Tailwind CSS

**Backend:** Java, Spring Boot, PostgreSQL

## Contributing

We welcome contributions from developers, designers, researchers, and the community. Explore our repositories, find an issue, and submit a pull request.
